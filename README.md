# Git Tutorial

## I. Các khái niệm
```
Repository (Kho chứa source code) - Repo
Branch (Nhánh) - master => main
Commit (mã hash, hash commit)
Pull Request (PR)
```




## II. Command:
```
git clone <repo_link>
git status
git branch // hien thi danh sach cac nhanh
git add . // Them file vao trang thai staged truoc khi commit
git commit -m "Create file danh_sach_log.txt"
git log --oneline -10 // Hien thi danh sach cac commit
git push origin main // Day code len repo github

git checkout -b <branch> // Tạo nhánh test1 và chuyển sang nhánh đó
git checkout <branch> // chuyen sang nhanh <branch>

git pull origin main // kéo code mới nhất về
```
